# AI-flashcard-generator
An AI-powered learning tool built for the Google Gemini Hackathon that automatically transforms study materials, notes, or topics into interactive flashcards using the Gemini API to optimize retention and study efficiency. 

## Key Features
* **Contextual AI Generation:** Uses Gemini to identify the most important concepts in a text.
* **One-Click Creation:** Converts paragraphs of text into Q&A pairs in seconds.
* **Smart Formatting:** Ensures cards are concise and easy to review.
* **Responsive Design:** Study on your desktop or mobile device.
